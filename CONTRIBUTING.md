# 🤝 Contribuer à AnimeFR Bot

Merci de vouloir contribuer à ce projet ! Voici comment participer.

## 📋 Comment contribuer

1. **Fork** le repository
2. **Clone** votre fork : `git clone https://github.com/votre-username/AnimeFR-Bot.git`
3. **Créez une branche** : `git checkout -b feature/ma-fonctionnalite`
4. **Faites vos modifications**
5. **Testez** que le bot démarre sans erreur
6. **Commit** : `git commit -m "feat: description de la fonctionnalité"`
7. **Push** : `git push origin feature/ma-fonctionnalite`
8. **Ouvrez une Pull Request**

## 🏷️ Convention de commits

| Préfixe | Description |
|---------|-------------|
| `feat:` | Nouvelle fonctionnalité |
| `fix:` | Correction de bug |
| `docs:` | Documentation |
| `style:` | Formatage (pas de changement de code) |
| `refactor:` | Refactoring |
| `perf:` | Amélioration de performance |
| `test:` | Ajout de tests |

## ⚠️ Règles importantes

- **Ne jamais commit de tokens ou clés API** dans le code
- Utilisez les variables d'environnement pour les secrets
- Testez votre code avant de soumettre une PR
- Respectez le style de code existant
- Commentez le code complexe en français

## 💡 Idées de contributions

- Nouvelles sources de streaming
- Nouveaux templates de posts
- Traductions (anglais, arabe, etc.)
- Amélioration de l'IA (meilleurs prompts)
- Tests unitaires
- Documentation

## 🐛 Signaler un bug

Ouvrez une **Issue** avec :
- Description du problème
- Étapes pour reproduire
- Comportement attendu vs observé
- Version du bot utilisée
- Logs d'erreur (sans tokens !)
